LoRaWAN Worker Tracking System Evaluation

Project Overview

A comprehensive evaluation of a LoRaWAN-based real-time worker tracking system for remote oil palm plantations in Sabah, Malaysia.

Project Objectives
  • Assess LoRaWAN network coverage in dense plantation environments
  • Evaluate GPS tracking accuracy under tree canopies
  • Test device performance and battery life
  • Analyze cellular signal strength for backhaul connectivity
